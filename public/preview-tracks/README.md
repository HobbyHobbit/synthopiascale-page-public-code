# Preview Tracks

This directory contains audio preview files for sample packs organized by product.

## Directory Structure
Each product has its own subdirectory for better organization:

```
preview-tracks/
├── phonk-vol1/
│   ├── kick-preview.wav
│   ├── loop-preview.wav
│   └── bass-preview.wav
├── progressive-house-vol1/
│   ├── melody-preview.wav
│   └── synth-preview.wav
├── signature-vol1/
│   ├── bass-preview.wav
│   └── drums-preview.wav
└── ... (other product directories)
```

## File Naming Convention
- Use descriptive names: `track-name.wav` or `track-name.mp3`
- Keep filenames web-friendly (no spaces, special characters)
- Use lowercase with hyphens
- Add "-preview" suffix for clarity

## Supported Formats
- **WAV**: High quality, larger file size
- **MP3**: Compressed, smaller file size, better for web streaming

## Usage
These preview tracks will be displayed in sample pack cards for users to listen before purchasing.
